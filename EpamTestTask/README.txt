Доброго дня!

Задача была написать консольную программу которая будет отслеживать задачи, добавлять сотрудников и проекты.
Так же по функционалу: бд должна была подключаться автоматически и методы должны быть покрыты тестами.

Реализовано было на мной взгляд 80% функционала программы. По логике программы не успел сделать возврат проектов
и привязанных сотрудников, не успел покрыть тестами, и не реализовывал автоматическую инициализацию бд.

По БД: по идее таблица создана и лежит в папке SQLite, там же лежит exe.шник для запуска командной строки.
Если что то вдруг пойдет не так необходимо будет заново инициализировать БД, и в самом коде указать
путь расположения папки sqlite. Приложение собирал через maven.

Запуск приложения:

1. Запустить командную строку sqlite.
2. в командной строке написать .open users.db

БД должна запустится, если этого не произошло:
1. удалить файл users.db
2. открыть командную строку sqlite
3. написать .open users.db
4. скопировать и вставить текст находящийся в файле "базаданных1"
5. PROFIT

Далее: 
3. открыть файл с приложением в среде разработки
4. в классе DataBase поменять путь расположения файла БД (я закоментил строчку)
5. добавить к зависимостям приложенный Jar файл, если уже стоит более новая версия JDBC то наверно сойдет,
но на всякий случай есть архив.
6. Нажать кнопку run.

Ну вроде все, надеюсь ничего не упустил. Всех благ!
